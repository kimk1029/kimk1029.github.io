Next.js 기반 포트폴리오 사이트. GitHub Pages에 정적 내보내기로 배포됩니다.

[![License: MIT](https://img.shields.io/badge/License-MIT-orange.svg)](https://opensource.org/licenses/MIT)

## Deploy (Next.js + GitHub Pages)

1. **로컬 실행**: `npm install` → `npm run dev`
2. **정적 빌드**: `npm run build` (결과물: `out/` 폴더)
3. **자동 배포**: `master` 또는 `main` 브랜치에 push하면 GitHub Actions가 빌드 후 Pages에 배포합니다.

**GitHub 저장소 설정**:  
Settings → Pages → Build and deployment → Source를 **GitHub Actions**로 선택해야 합니다.
